# Saxophone Performance Website

This repository contains the source files for my professional saxophone performance website.

The site promotes live saxophone services for weddings, private events, cocktail hours, and special occasions. Visitors can learn about my style, view performance videos, and contact me for booking inquiries.

## Website Features

* Information about live saxophone performance 
* Embedded performance videos
* Photo gallery 
* Contact information for booking

## Technologies Used

* HTML
* CSS
* Static image and video embeds

## Deployment

The site is hosted using Cloudflare Pages and automatically deploys when updates are pushed to the repository.

## About the Performer

I provide live saxophone performances for weddings, receptions, cocktail hours, and private events. My style blends smooth jazz, pop, and atmospheric improvisation to create an elegant musical backdrop for memorable occasions.

For booking inquiries, please visit the contact page on the website.
